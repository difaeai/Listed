# **App Name**: SalesConnect

## Core Features:

- User Authentication: User authentication with distinct sign-up/sign-in flows for companies and sales professionals.
- Ad Management: Dashboard for companies to create and manage offer ads targeted at sales professionals.
- Offer Listing: Listing of available offers for sales professionals. Sales professionals can view ads and register a sale with the company.

## Style Guidelines:

- Primary color: A professional and trustworthy blue (#3498db).
- Secondary color: A clean, neutral gray (#ecf0f1) for backgrounds and subtle elements.
- Accent: A vibrant green (#2ecc71) for calls to action and positive feedback.
- Clean and intuitive layout with clear sections for ads, user profiles, and dashboards.
- Use professional and recognizable icons to represent different actions and categories.
- Subtle transitions and animations to provide feedback and guide the user through the app.
